面试题14-剪绳子

发表于 2019-11-27 | 分类于 剑指offer

给你一根长度为n的绳子，请把绳子建成m段（m、n都是整数，并且m > 1), 每段绳子的长度记为k[0],k[1],k[2],k[3],请问他们相乘的最大长度是多少？例如，绳子为18的时候可以切为2，3，3三段，此时得到的最大乘积是18

动态规划解法

如果满足四个条件，则可用动态规划解题

  1. 求一个问题的最优解
  2. 整体问题的最优解以来各个子问题的最优解
  3. 大问题可以分解成若干个小问题，这些小问题之间还有相互重叠的更小的子问题
  4. 从上往下分析问题，从下往上求解问题

求解思路：一条长度为n绳子，如果想要剪一刀的话，则有n-1种选择，所以，f(n)=max(f(i) * f(n - i))，其中 1 < i < n

这是一个从上而下的递推公式，我们从下往上计算，用一个一维数组记录下最优解，因为绳子长为1，2，3的时候，我们可以直接return，但是当绳子>3的时候，小于3的部分局部最优解是1，2，3，也就是不进行切割。

时间复杂度：O（n^2）

空间复杂度: O (n)

贪心算法

贪心算法的基本思路是从问题的某一个初始解出发一步一步地进行，根据某个优化测度，每一步都要确保能获得局部最优解。每一步只考虑一个数据，他的选取应该满足局部优化的条件。若下一个数据和部分最优解连在一起不再是可行解时，就不把该数据添加到部分解中，直到把所有数据枚举完，或者不能再添加算法停止

当n>=5的时候，我们可以证明2（n-2）>n && 3(n-3)>n.也就是说，当绳子剩下的长度大于或者等于5的时候，我们就把它剪成长度为3或者2的绳子段，另外，当n>=5时，3(n-3)>=2(n-2),因此我们应该尽可能的多剪长度为3的绳子.
